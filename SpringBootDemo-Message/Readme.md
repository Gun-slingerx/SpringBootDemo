### 调用阿里大鱼发送短信
    SendSms接口是短信发送接口，支持在一次请求中向多个不同的手机号码发送同样内容的短信。
    SendBatchSms接口是短信批量发送接口，支持在一次请求中分别向多个不同的手机号码发送不同签名的短信。手机号码等参数均为JSON格式，字段个数相同，一一对应，短信服务根据字段在JSON中的顺序判断发往指定手机号码的签名。
    有回执的短信（例如 退订回复T），此类短信需要获取回执内容
